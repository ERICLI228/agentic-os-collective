## UX-01: 打开里程碑 → 3秒内看到摘要卡片

**操作步骤:**
1. 打开 http://localhost:5001/dashboard
2. 点击左侧任意 TK 里程碑（如 MS-0, MS-1, MS-2）
3. 观察详情区顶部

**预期结果:**
- 显示里程碑摘要卡片（状态图标 ✅/⚠️/❌）
- 卡片包含一句话结论 + 核心数字
- 显示"查看详情"按钮
- 详细内容默认折叠

**当前状态:** ⏳ 待测试

## UX-02: 点击"查看详情"展开详细内容

**操作步骤:**

3. 观察详细内容显示/隐藏

**预期结果:**
- 点击后展开详细数据面板
- 再次点击折叠
- 展开/折叠动画流畅

**当前状态:** ⏳ 待测试

## UX-03: DM-0 触发审核并查看实时日志

**操作步骤:**
1. 切换到 Drama 业务线
2. 点击 DM-0 里程碑
3. 在详情区找到"重新审核"按钮并点击
4. 观察日志面板

**预期结果:**
- 显示实时日志面板（review-log-panel）
- 日志逐条显示（⏳→📖→📝→🎬→⏱️→🧠→✅）
- 每条日志间隔约400ms
- 完成后显示评分（X/10）
- 页面自动刷新

**当前状态:** ⏳ 待测试

## UX-04: 审核完成后查看四维可折叠卡片

**操作步骤:**
1. DM-0 审核完成后（或已有审核数据）
2. 在审核结果区找到四维卡片
3. 点击卡片展开/折叠
4. 查看问题列表和改进建议

**预期结果:**
- 看到4张维度卡片（📝编剧/🎬场景/⏱️节奏/🧠逻辑）
- 每张卡片有分数（低分红色、高分绿色）
- 点击卡片展开问题列表（⚠️）和建议列表（💡）
- 再次点击折叠

**当前状态:** ⏳ 待测试

## UX-05: 剧本保存后重审引导

**操作步骤:**
1. 打开任意剧集详情（如 DM-0 → 某集）
2. 点击"页内预览"
3. 修改剧本内容
4. 点击"保存并重新审核"

**预期结果:**
- 保存成功后弹出确认框"是否立即重新审核？"
- 显示上次得分（如有）
- 提供"重新审核"和"稍后"两个按钮
- 点击"重新审核"后自动触发审核
- 完成后显示新旧分数对比

**当前状态:** ⏳ 待测试
