# ⚖️ AI短剧合规检查清单

> 基于广电总局"AI魔改"专项治理标准

---

## 🔴 绝对禁止 (红线)

| 项目 | 禁止内容 | 风险等级 |
|------|----------|----------|
| 歪曲原著 | 严重违背原作精神内核，篡改核心价值观 | 🔴 高 |
| 低俗恶搞 | 低俗化改编，为流量无底线戏谑 | 🔴 高 |
| 暴力血腥 | 过度渲染剖腹、挖心等血腥场面 | 🔴 高 |
| 恶俗色情 | 任何色情擦边内容 | 🔴 高 |
| 歪曲历史 | 篡改历史人物和事件 | 🔴 高 |

---

## 🟡 需要改编 (黄线)

| 原著情节 | 问题 | 改写建议 |
|----------|------|----------|
| 宋江招安 | 愚忠价值观 | 改为"识破奸臣阴谋，保护梁山兄弟" |
| 李逵滥杀 | 滥杀无辜 | 改为"只杀官兵，不伤百姓" |
| 潘金莲 | 女性形象物化 | 增加合理动机，反对压迫 |
| 征方腊 | 兄弟惨死 | 减少惨烈描写，注重兄弟情义 |
| 吃人肉 | 血腥暴力 | 删除或远景暗示 |

---

## 🟢 推荐方向 (安全)

| 类型 | 说明 |
|------|------|
| 兄弟情义 | 突出梁山好汉之间的义气 |
| 反抗精神 | 强调"替天行道"的正面意义 |
| 智慧谋略 | 智取生辰纲、三打祝家庄等 |
| 英雄本色 | 武松打虎、鲁智深倒拔垂杨柳 |
| 正义审判 | 拳打镇关西、怒杀西门庆 |

---

## 📝 制作前检查

- [ ] 确认剧本不涉及上述禁止内容
- [ ] 确认角色形象为AI原创，无真人相似
- [ ] 确认不参考受版权保护的影视作品
- [ ] 确认已添加AI生成标识

---

## 📝 制作后检查

- [ ] 视频画面无过度血腥暴力
- [ ] 角色对话无低俗言语
- [ ] 整体价值观符合主流价值观
- [ ] 无政治敏感内容
