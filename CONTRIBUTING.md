Contributing to Agentic OS Collective

- Start by forking or creating a feature branch if contributing to this public repo.
- Ensure tests pass locally before opening a PR.
- Follow the code style and testing conventions used by the project.
- Invite collaborators via GitHub: add collaborators to the repository settings or add them to CODEOWNERS for automatic review.
- Submit a PR with a descriptive title and include test coverage where applicable.
- Reach out to maintainers for any questions.

Guidelines for inviting collaborators:
- On GitHub, go to Settings > Manage access > Invite a collaborator.
- Enter the collaborator's GitHub username and grant appropriate access.
- Consider using teams for scalable access control.
